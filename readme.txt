Програма для анализа привлекательности цены акций компании.

С записью данных в инфо файл

файл: src/main/resources/StocksInfo.txt: содержит информацию о компаниях, которая будет анализироваться.

когда находитесь в папке с файлами проекта, откройте терминал;
передайте строку в консоль, для запуска программы:

java -jar target/AdditionCalculationOfAverageValues-1.0-SNAPSHOT-jar-with-dependencies.jar

если в терминале появились символы вместо букв, то запустите программу используя эту строку

java -Dfile.encoding=UTF-8 -jar target/AdditionCalculationOfAverageValues-1.0-SNAPSHOT-jar-with-dependencies.jar
